Test Administrator Web-App

How to use:

  1. Download the question template using the Download Template menu button.
  2. Open the template in excel.
  3. Add questions to the template using the appropriate format already shown.
  4. Save the document as a .csv file.
  5. Click on the Question Admin menu button.
  6. Click "Choose File" and select the csv file.
  7. An alert will show if the file was successfully uploaded.
  8. Click Load to see the current questions in the database.
  9. To see a sample test made of the current questions, click Start Test.
